# 樟林古港文旅服务小程序

这是一个基于原版微信小程序架构构建的文旅项目，面向游客提供以下核心能力：

- 景点智能讲解
- 地域文化科普
- 全域游玩导览
- 本土特色风物推介

## 页面结构

- `pages/index`：首页，整合品牌介绍、功能入口、推荐景点与主题路线
- `pages/guide`：智慧导览页，提供半日游、一日游、亲子研学等路线
- `pages/culture`：文化科普页，按专题介绍红头船、侨乡文脉与建筑风貌
- `pages/specialty`：特产风物页，展示在地风味、文创与潮汕工艺体验
- `pages/scenic-detail`：景点详情页，承载重点景点的智能讲解内容

## 使用方式

1. 打开微信开发者工具
2. 导入当前目录作为小程序项目
3. 如需真机或正式预览，请将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID

## AI助手「阿樟」接入说明

### 能力定位

「阿樟」是樟林古港文旅小程序的AI旅游管家，基于Tavily实时搜索API与本地知识库构建，提供以下核心服务：

- **景点智能讲解**：针对红头船文化展示点、古港遗址、侨乡建筑群等景点提供深度讲解
- **文化知识问答**：回答关于红头船海贸历史、侨乡文脉、古港建筑等方面的咨询
- **游览路线推荐**：根据游客时间和兴趣推荐半日游、一日游或亲子研学路线
- **在地特色推介**：介绍潮汕工夫茶、牛肉火锅、英歌舞等本地风物

### 技术实现

AI能力由 `utils/ai-service.js` 提供支持：

```javascript
const { askAI } = require('./utils/ai-service.js');

// 调用示例
askAI('红头船为什么是红色的？', {
  context: { currentPage: 'scenic-detail', scenicId: 'ship-museum' }
}).then(result => {
  console.log(result.answer);
  console.log(result.source); // 'local' | 'tavily' | 'local-fallback'
});
```

### 接入流程

1. 获取 Tavily API Key（已配置：`tvly-dev-***ATDa`）
2. 在 `utils/ai-service.js` 中配置 `TAVILY_API_KEY`
3. 小程序端通过 `askAI()` 函数调用AI服务
4. 支持设置 `useLocalFirst: true` 优先使用本地预设问答
