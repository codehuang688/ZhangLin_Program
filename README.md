# 樟林古港文旅服务小程序

这是一个基于原生微信小程序结构搭建的静态演示项目，面向游客提供以下核心能力：

- 景点智能讲解
- 地域文化科普
- 全域游玩导览
- 本土特色风物推介

## 页面结构

- `pages/index`：首页，整合品牌介绍、功能入口、推荐景点与主题线路
- `pages/guide`：智慧导览页，提供半日游、一日游、亲子研学等路线
- `pages/culture`：文化科普页，按专题介绍红头船、侨乡文脉与建筑风貌
- `pages/specialty`：特色风物页，展示在地风味、文创与民俗体验
- `pages/scenic-detail`：景点详情页，承载重点景点的智能讲解内容

## 使用方式

1. 打开微信开发者工具。
2. 导入当前目录作为小程序项目。
3. 如需真机或正式预览，请将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID。

## 数据说明

项目演示数据集中在 `data/site.js`，后续可继续接入：

- 后端接口数据
- 音频讲解资源
- 地图导航能力
- 活动资讯与游客服务信息
